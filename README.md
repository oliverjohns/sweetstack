# $weet$tack - T3 with Supabase Auth + Drizzle ORM

Project bootstrapped with `create-t3-app`. Uses Drizzle ORM instead of Prisma and Supabase Auth instead of NextAuth.

- [Next.js](https://nextjs.org)
- [Drizzle ORM](https://drizzle.team/)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)

- [T3 Documentation](https://create.t3.gg/)
